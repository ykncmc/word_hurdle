# Word Hurdle

A Wordle-style mobile game built using Flutter and Provider.

## 📱 Features
- 5-letter word guessing game
- 6 attempts system
- Dynamic UI updates using Provider
- Random word generation

## 🛠 Tech Stack
- Flutter
- Dart
- Provider (State Management)

## 🚀 How to Run
```bash
flutter pub get
flutter run
